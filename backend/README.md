# 🏠 Roomify Backend

## 📌 Overview
Backend of the Roomify PG management system built using Node.js and Express.js. It handles APIs, authentication, database operations, and core business logic.

---

## ⚙️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Cloudinary

---

## 🚀 Features
- User Authentication (Login/Register)
- PG Listing Management
- Booking/Request System
- Receipt Generation
- Reviews & Ratings
- Image Upload & Management

---

## ▶️ Run Server
cd backend  
npm install  
npm start  

Runs on: http://localhost:5000

---

## 🔐 Environment Setup
Create a `.env` file and add:
- MONGO_URI
- JWT_SECRET
- CLOUDINARY credentials
